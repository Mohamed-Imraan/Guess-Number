<h1 align="center">🎲 Lucky Game – Number Guessing 🎮</h1>

> Welcome to **Lucky Game**, a fun number guessing project built with **HTML, CSS, and JavaScript**.  
> The player tries to guess the secret number between 1 and 10, with a starting score of 10.

## 🔗 Live demo : https://guess-number-lucky-game.vercel.app/

## 🚀 Features
- 🎮 Random number generation between 1–10
- 🖥️ Interactive UI with gradient background
- ✅ Feedback messages (Correct / Wrong)
- 📉 Score decreases with wrong guesses
- 🔄 Reset by refreshing the page

## 🛠️ Tech Stack
- **HTML5** – Game structure
- **CSS3** – Styling and animations
- **JavaScript (ES6)** – Game logic
